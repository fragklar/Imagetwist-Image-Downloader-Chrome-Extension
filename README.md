# Imagetwist-Image-Downloader-Chrome-Extension
Ein kompaktes Browser-Tool (Manifest V3) zum automatischen Hinzufügen von Download-Buttons auf Bilder-Hosting-Webseiten. Ermöglicht das sofortige Speichern von Bildern mit der Klasse ipsImage_thumbnailed über die Chrome-Downloads-API. Lizenziert unter MIT von fragklar.de (2026)
